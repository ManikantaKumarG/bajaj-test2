# SRM Full Stack - BFHL Solution

This project includes:
- Node.js + Express backend with `POST /bfhl`
- Single-page frontend for entering edges and viewing the response

## Run locally

```bash
npm install
npm start
```

Open:
- `http://localhost:3000` (frontend)
- `http://localhost:3000/bfhl` (API route - POST)

## Important before submission

Update these fields in `server.js` with your real details:
- `user_id`
- `email_id` (college email)
- `college_roll_number`

## Hosting

You can deploy this directly on Render/Railway/Vercel (Node app).  
After deployment, share:
- Hosted backend URL (`<your-url>/bfhl`)
- Hosted frontend URL
- Public GitHub repository URL
